bmi_calculator

  • Githubのページはこちら
  • RubyGemsのページはこちら

身長、体重からBMIを計算します。
利用できる単位は下記になります。
  • m（メートル：身長）
  • cm（センチメートル：身長）
  • inch（インチ：身長）
  • feet（フィート：身長）
  • kg（キログラム：体重）
  • g（グラム：体重）
  • pound（ポンド：体重）
gemのインストールは下記のコマンドを実行します。

            
              $ gem install bmi_calculator
利用方法についてですが、実際に利用可能な組み合わせを、下記ソースで説明します。

            
              require 'bmi_calculator'

              # 馴染み深い単位での計算をします。
              # 第一引数が身長、第二引数が体重です。

              # m（1.7メートル）、kg（60キログラム）でBMIを計算します。
              bmi = BmiCalculator.calc_m(1.7, 60)
              # cm（170センチメートル）、kg（60キログラム）でBMIを計算します。
              bmi = BmiCalculator.calc_cm(170, 60)
              # cm（170センチメートル）、g（60000グラム）でBMIを計算します。
              bmi = BmiCalculator.calc_cm_g(170, 60000)


              # ここからは、ヤードポンド法で計算します。

              # feet（5フィート）、inch（7インチ）、ポンド（130ポンド）でBMIを計算します。
              bmi = BmiCalculator.calc_yp(5, 7, 130)
              # feet（5.5フィート）、ポンド（130ポンド）でBMIを計算します。
              bmi = BmiCalculator.calc_feet_yp(5.5, 130)
              # inch（67インチ）、ポンド（130ポンド）でBMIを計算します。
              bmi = BmiCalculator.calc_inch_yp(67, 130)
上記のように、簡単に計算ができます。
