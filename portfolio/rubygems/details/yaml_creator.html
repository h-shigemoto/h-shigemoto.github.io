yaml_creator

  • Githubのページはこちら
  • RubyGemsのページはこちら

YAMLファイルを作成します。
対応しているものは下記になります。
  • 配列
  • ハッシュ
  • JSON
配列は入れｺになっている場合、平坦化した後、YAMLとしてファイルに記述します。

gemのインストールは下記のコマンドを実行します。

            
              $ gem install yaml_creator
利用方法についてですが、下記ソースで説明します。

            
              require 'yaml_creator'

              # 配列からYAMLファイルを作成します。
              array = ["a", "b", "c", "d"]
              YamlCreator.from_array("書き込みファイルパス", array)
              # 囲み文字を指定する場合、第三引数に指定できます。
              YamlCreator.from_array("書き込みファイルパス", array, "'")

              # ハッシュからYAMLファイルを作成します。
              hash = { a: => "valueA", b: => "valueB", c: => "valueC" }
              YamlCreator.from_hash("書き込みファイルパス, hash)
              # 囲み文字を指定する場合、第三引数に指定できます。
              YamlCreator.from_hash("書き込みファイルパス", hash, "'")

              # JSONからYAMLファイルを作成します。
              json = "{\"a\":\"valueA\",\"b\":\"valueB\",\"c\":\"valueC\"}"
              YamlCreator.from_json("書き込みファイルパス", json)
              # 囲み文字を指定する場合、第三引数に指定できます。
              YamlCreator.from_json("書き込みファイルパス", json, "'")
上記のように簡単にYAMLファイルが作成できます。
書き込みファイルパスを指定せずにメソッドを呼び出した場合、ファイル名は、
「%Y%m%d%H%M%S%3N.yml」になります。
