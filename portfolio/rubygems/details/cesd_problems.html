cesd_problems

  • Githubのページはこちら
  • RubyGemsのページはこちら

CES-D（うつ病自己評価評価尺度）の問題、解答及び点数配点をセット（ハッシュ）で定義したものです。
準備した言語は、下記の2つになります。
  • 日本語
  • 英語

gemのインストールは下記のコマンドを実行します。

            
              $ gem install cesd_problems
取得方法は複数あり、問題文と点数配点のみ、解答のみといった取得も可能になります。
勿論、質問、解答及び点数配点を一度に取得することもできます。
実際に下記、コードを交えて説明します。

            
            require 'cesd_problems'

            # まずは、日本語で取得します。

            # 日本語の問題、点数配点等全てを取得する。
            all = CesdProblems.get_ja_problems

            # ここからは、個別に取得していきます。
            # 日本語の問題と、それぞれの配点のみ取得する。
            problems = CesdProblems.get_ja_problems(CesdProblems::PROBLEMS)
            # 日本語の解答（4択）のみ取得する。
            answers = CesdProblems.get_ja_problems(CesdProblems::ANSWERS)
            # 判定基準点数のみ取得する。
            criteria = CesdProblems.get_ja_problems(CesdProblems::CRITERIA)


            # 上記と同様に、英語でも取得していきます。

            # 英語の問題、点数配点等全てを取得する。
            all = CesdProblems.get_en_problems
            # 英語の問題と、それぞれの配点のみ取得する。
            problems = CesdProblems.get_en_problems(CesdProblems::PROBLEMS)
            # 英語の解答（4択）のみ取得する。
            answers = CesdProblems.get_en_problems(CesdProblems::ANSWERS)
            # 判定基準点数のみ取得する。
            criteria = CesdProblems.get_en_problems(CesdProblems::CRITERIA)


            # 日本語と英語、一度に全て取得してみます。
            all = CesdProblems.get_all_lang
            # 日本語と英語の問題と、それぞれの配点のみ取得する。
            problems = CesdProblems.get_all_lang(CesdProblems::PROBLEMS)
            # 日本語と英語の解答（4択）のみ取得する。
            answers = CesdProblems.get_all_lang(CesdProblems::ANSWERS)
            # 判定基準点数のみ取得する。
            criteria = CesdProblems.get_all_lang(CesdProblems::CRITERIA)

取得結果は、基本的にハッシュで返却しています。
どう返却しているかは、こちらをご確認ください（出力したほうが早いかもしれないので、pとかppとかで確認でもいいと思います）。

4択の解答と、各問題文に対する配点は、基本的にラジオボタンが多いと思って、左から順に設定しています（それぞれ配列で定義しています）。
1番目の選択肢が1番目の配点と対応、2番目の選択肢が2番目の配点と対応、といった感じです。
そこらへんの兼ね合いもあって、問題と配点をセットで返却したりもしています。

実際に内部的にロジックはほとんどなく、配列で定義したCES-Dの内容を返却しているだけです。
なので、GitHubを読めばすぐわかるレベルだと思います。
