easy_zip

  • Githubのページはこちら
  • RubyGemsのページはこちら

対象ファイルに文字列を書き込み、GZipに圧縮します。
また、GZipファイルから文字列を読み込みます。
# 今後普通のファイルをGZipにしたりする処理も入れていく予定です。

gemのインストールは下記のコマンドを実行します。

            
              $  gem install easy_zip
では使い方です。
すごく簡単です。。。。

            
              # GZipファイルの全行を読み込みます。
              lines = EasyZip::Gzip.read_lines('読み込みGZipファイルパス')
              # GZipファイルに単一の内容を書き込みます。
              EasyZip::Gzip.write_lines('書き込みGZipファイルパス', '書き込む内容')
              # GZipファイルにすべての内容を書き込みます。
              EasyZip::Gzip.write_lines('書き込みGZipファイルパス', ['1行目', '2行目', '3行目'])
上記のように、簡単にテキストをGZipファイルに書き込むことができます。
